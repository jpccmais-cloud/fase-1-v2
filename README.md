# Projeto – Sistema Web de Petshop (Fase 1)

**Metas específicas:**
1. **Estruturar as páginas básicas** com semântica HTML5 (header, main/body, footer).
2. **Exibir produtos por categoria** (3 categorias, 2 produtos por categoria) com **imagem, descrição e valor**.
3. **Listar serviços oferecidos** com **descrição, valor** e indicação de **tele-busca (Sim/Não)**.
4. **Inserir navegação** entre as páginas por meio de links no cabeçalho e rodapé.
5. **Preparar o projeto para publicação** no GitHub Pages (fase 1 sem CSS/JS).

---

## Estrutura das Páginas
- `index.html`: página inicial com apresentação e destaques.
- `produtos.html`: catálogos com três categorias (Rações, Brinquedos, Higiene), dois itens por categoria, cada item com imagem/descrição/preço.
- `servicos.html`: lista de serviços com descrição, preço e **Tele-busca: Sim/Não**.
- `contato.html`: informações de contato e **formulário ilustrativo** (sem funcionalidade nesta fase).

Cada página contém:
- **Header** com nome do petshop e navegação.
- **Footer** com créditos de autoria, contato e navegação.
- **Sem CSS/JS** (exigência da fase 1).
